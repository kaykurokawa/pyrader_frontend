How to Use

1. In the first dropdown, pick "Block" or "Price/Volume".

2. In the next dropdown, select what coin you want to look up (BTC, LTC, ETH, etc..)

3. Follow the arrows to select the inputs for the request.

The following is a brief explanation of the drop down menus:

Units: What units you want to see the data in terms of (USD, BTC, EUR).

Exchange: Request which exchange you want to get the data from,
pick "Aggregated" for an average of all exchanges.

Datatype: The type of block data:

  • Block size (block_size_bytes)
  • Block Time (block_time)
  • Difficulty (difficulty)
  • Height (height)
  • Number of Tickets (num_tx)
  • Total Transfer Volume (total_transfer_volume)
  • Total Transfer Voluime in USD (total_transfer_volume_usd)

Averaging: The frequency of the intreval of data point.

  • 5 minutes:
    Data collected every 5 minutes.
  • 1 hour:
    Data collected every hour.
  • daily:
    Data collected every day.
  • No Averaging:
    Data is collected at random intervals and displayed in scatter plot.

5. Hit the Submit button to get the data. Hit Reset to reset the drop downs.

6. You can add multiple data types to the graph.

7. Use the range slider or date input on the graph to navigate your chart:

8. On the table click to remove from graph.
